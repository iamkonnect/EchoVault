# Sample Images Folder

**Purpose**: Store sample images for demo data, thumbnails, and testing.

## Usage Examples

```
├── demo-dashboard.json → "thumbnail": "/assets/samples/dashboard-thumb.jpg"
├── demo-live.json     → "thumbnail": "/assets/samples/live-thumb.jpg"  
├── demo-shorts.json   → "thumbnail": "/assets/samples/shorts-thumb.jpg"
└── demo-wallet.json   → "avatar": "/assets/samples/wallet-avatar.jpg"
```

## Supported Formats
- PNG, JPG, WebP, SVG
- Max recommended size: 512x512px
- Name files descriptively (e.g. `dashboard-hero.jpg`)

**Upload your images here via VSCode or File Explorer!**
